## 第 4 页 · 引擎分层架构（30 秒）

回来讲架构。代码分两层——**base** 和 **game**。

base 是渲染引擎，和游戏无关：Model、Shader、Framebuffer、Texture、Camera、Skybox、OBJ Loader。全部 RAII 封装——构造函数 `glGen*`，析构 `glDelete*`，从根上杜绝句柄泄漏。

game 是游戏逻辑：瞄准训练、相机控制、粒子、音频、武器、靶子、计分排行。game 不耦合引擎实现细节，依赖接口。

依赖的第三方库列在这里——OpenGL 3.3、GLFW、GLM、Dear ImGui、stb_image、miniaudio。都是课程推荐的基础库，不提供高层功能。

## 第 5 页 · 多 Pass 渲染管线（45 秒）

这是最关键的一页——**多 Pass 渲染管线**。一帧不是一遍画完的。

流程：
1. **Pass 1**：方向光阴影深度图，2048×2048 的 FBO。
2. **Pass 1b**：聚光灯阴影深度图，同样的分辨率。
3. **Pass 2 主渲染**：Blinn-Phong 画场景 → 天空盒 → 粒子加法混合 → 枪械视角模型 → 开镜遮罩 → HUD。

三个设计要点：
- **几何复用**：`drawScene()` 被两个深度 Pass 和主 Pass 共用，一个 `withMaterial` 布尔决定是否绑材质。一份几何三处渲染。
- **枪不穿模**：画枪前 `glClear(GL_DEPTH_BUFFER_BIT)` 清空深度，独立近平面 0.005，枪永远在最前面——这是 FPS 经典做法。
- **天空盒置后**：顶点着色器 `.xyww` 强制深度为 1.0，`GL_LEQUAL` 只填未被覆盖的像素，省了 overdraw。

## 第 6 页 · 自研 OBJ 模型加载器（40 秒）

OBJ 加载是我们自己写的——没用 tinyobjloader，没用 Assimp。

三个核心：
1. **解析四种面格式**——"1"纯位置、"1/2"位置/UV、"1//3"位置/法线、"1/2/3"全属性。数斜杠位置判断属性是否存在，纯字符串解析，零依赖。
2. **顶点去重**——用 `unordered_map<Vertex, uint32_t>` 做去重，自定义 hash 异或折叠，`operator==` 三元组判等。O(1) 查重，EBO 索引复用省显存。
3. **三角化**——任意 N 边形用三角扇覆盖。

亮点是**自产自销闭环**——F1 导出场景 OBJ，开场变形球动画的每一帧也是用这个导出器生成的，再用自己的加载器读回来播放。自己写的格式，自己吃自己的输出，不用信任外部工具。

## 第 7 页 · 自研 NURBS / B 样条曲线（30 秒）

NURBS 曲线也没用 GLU 的 NURBS 接口——手写 Cox-de Boor 基函数递归。

代码就是标准递推公式：P 为 0 时判断 t 是否在区间内，P > 0 时左递归加右递归。做完整套除零保护。

三次周期 B 样条：前三个控制点接末尾，形成无缝闭环——**wild 模式里那几个球不会突然跳一下**。

诚实声明：权重均取 1，本质是均匀 B 样条，但框架支持和 NURBS 完全一致。

wild 模式**【切到 Wild，Tab 俯瞰展示 NURBS 轨迹】**——三球各有偏移曲线加上错开相位，轨迹各异。

## 第 8 页 · 双阴影贴图 + 3×3 PCF（40 秒）

阴影我们做了两套——方向光 ortho 投影 + 聚光灯 perspective 投影，各 2048×2048。

技术步骤：
1. 深度 FBO，CLAMP_TO_BORDER 白边——框外的物体不被错误遮挡。
2. 深度 Pass 用 `glCullFace(GL_FRONT)` 剔除正面，缓解自阴影（peter-panning）。
3. Fragment 里做 3×3 PCF，9 个邻域采样取平均得到软阴影系数。

另外，多光源体系在同一着色器里——方向光 + 3 点光 + 2 聚光（含手电筒，**L 键**开关）+ 半球环境光。背光墙面不纯黑，ImGui 面板实时编辑。
**【F1 面板 → 勾选/取消 Directional Shadows，对比影子】**

## 第 9 页 · 碰撞检测 + 后室交互门（45 秒）

碰撞检测是三件套：
- **球-AABB**：玩家跟障碍物的碰撞——最近点 clamp 求距离，分轴解算先 X 后 Z，贴墙滑动不卡死。
- **球-球**：Wild 模式里两个靶子近了互斥推开。
- **射线-球**：射击命中的解析判别式，取最近正根。

重点讲**后室交互门系统**——这个完美对应"功能交互真实性"加分项：
1. **局部空间检测**：射线反变换到门的局部坐标系做 ray-AABB，跟门旋转角度无关。
2. **弹簧阻尼动画**：θ'' = -kθ - cθ'，命中给角初速度，门自然摆动到 90°。
3. **动态碰撞体积**：每帧按开合角度重建 AABB——开 ≥85° 碰撞拿掉，可进入后室。
**【演示走到后墙 → 射门 → 走进去 → 打金靶】**

## 第 10 页 · 射击反馈与视觉特效（25 秒）

这页展示视觉细节，快速过：

- **三层粒子爆炸**：CPU 模拟 + GL_POINTS 加法混合渲染——橙核、黄火花、暗红余烬。
- **弹簧阻尼后座力**：F = -kx - cv，开枪给冲量自动弹回。
- **狙击开镜**：FOV 缩到 15° 放大，全屏遮罩着色器把镜外区域涂黑加十字线。
- **即时反馈**：命中飘字世界到屏幕投影、动态准星 80px/s 弹回、命中标记 × 0.18 秒淡出。
