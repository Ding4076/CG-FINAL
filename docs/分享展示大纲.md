# AimTrainer 分享展示大纲

> 约 5–8 分钟讲 + 2 分钟演示。以**自研引擎 + 图形学技术亮点**为主线，不罗列菜单。

## 一、一句话定位（15 秒）

> "基于自研 OpenGL 引擎的三维射击训练游戏。不用 Unity/Unreal，不用 tinyobjloader，不用 GLU——全部手写。"

## 二、首先给看（15 秒演示）

双击 exe，展示**开场 OBJ 序列动画**（变形球，顶点逐帧变化）→ 自动进入游戏。
讲：**"这个开场是自研 OBJ 序列动画——30 帧 .obj 从磁盘连续读取，每帧网格不同，不是变换，是真几何变化。满足基本要求 7。"**

## 三、核心技术栈（30 秒）

| 层 | 内容 | 自研/第三方 |
|---|---|---|
| 渲染 | OpenGL 3.3 + GLSL | **手写 shader**（Blinn-Phong, shadow depth, scope mask） |
| 模型 | OBJ 解析/导出、6 种体素 | **手写 OBJ loader**（v/vt/vn/f 三种格式），替代 tinyobjloader |
| 曲线 | NURBS（Cox-de Boor） | **手写**，不用 GLU NURBS 接口 |
| 碰撞 | sphere-AABB, sphere-sphere | **手写**，滑动分离 |
| UI | Dear ImGui | 第三方（本课允许） |
| 音频 | miniaudio | 第三方 |

> 要点：**"框架从零搭建——source/base/ 是自研引擎，包括相机/材质/帧缓冲/天空盒/纹理等基础组件。"**

## 四、基本要求 7 项（快扫，1 分钟）

手指拨动、让观众看到：

1. **体素建模**：F1 面板里 6 种图元（球/立方体/圆柱/圆锥/多面棱柱/多面棱台）→ 可选 initPrimitives 展示
2. **OBJ 导入导出**：开场动画 = 连续 OBJ 读取；F1 → Export scene to .obj = 导出
3. **材质 + 纹理**：Blinn-Phong 光照 + rock/wall 贴图（地面/障碍物/门不同材质）
4. **几何变换**：靶子 scaling、Orbit 旋转、门弹簧阻尼旋转
5. **多光源编辑**：F1 面板实时调 dir/point×3/spot/手电位置/颜色/强度/锥角
6. **摄像机漫游**：Tab 切 FPS/Orbit，FPS 用 WASD+鼠标，Orbit 右键拖/滚轮/中键
7. **OBJ 序列动画 + 截图**：开场已完成；F2 截图当场存

## 五、加分亮点（重点讲，2 分钟）

### ⭐ NURBS 曲线（额外 1）
- 自研 Cox-de Boor 基函数，periodic 闭合曲线**无缝循环**
- **用在游戏里**（不是独立查看器）：Wild 模式 3 个球沿 NURBS 运动，球-球 sphere-sphere 互斥

> 演示：F1 → 切 Wild 模式 → Tab 切 Orbit 俯视，看 3 条曲线轨迹

### ⭐ 实时阴影（额外 3）
- 双 shadow map：directional light（ortho）+ spotlight（perspective）
- PCF 3×3 软阴影
- F1 可开关影子，当场对比

> 演示：F1 面板 Directional Light → 勾选/取消 Shadows，看地面影子变化

### ⭐ 碰撞检测（额外 2）
- **玩家** sphere-AABB：vs 栏杆/板条箱/墙壁，轴分离滑动（沿墙走，不卡死）
- **靶子互斥** sphere-sphere：Wild 模式球不重叠
- **动态门**：关门 = AABB 激活（不可通过），开门 = AABB 移除（可通过）
- 平台/坡道高度场：玩家可走上坡到后方高台

### ⭐ 完整三维游戏 + 交互真实性（额外 5 + 7）
- 双模式（Standard/Wild）+ 双武器（手枪/狙击）+ 计分 + 排行榜
- **交互门**（对应要求.txt PUBG 例子）：
  - 关门不可通过（碰撞体激活）
  - **射击开门**（弹簧阻尼动画，平滑旋转 90°）
  - 走进后室 → 金色大靶 +3 分奖励 + 专属粒子爆炸
  - 再射关门

> 演示：走到后墙门 → 射开门 → 走进去 → 打金色靶 → 看到粒子 + 分数跳变

## 七、演示流程（2 分钟）

> **按这个顺序走，别跳。**

1. **启动 exe** → 观众看 OBJ 开场动画（变形球）→ 自动进游戏
2. **手枪打两个靶** → 展示粒子+浮字+准心跳动+音效
3. **切狙击 `2` → 右键开镜 → 打靶** → 展示开镜遮罩+不同音效
4. **走到后墙 → 射开门 → 走进去 → 打金色大靶** → 粒子+ "+3" 分数
5. **F1 面板 → 切 Wild 模式 → Tab 俯瞰** → 展示 NURBS 曲线轨迹+球互斥
6. **F1 → 开关 Directional Shadows** → 对比影子有无
7. **F1 → Export scene to .obj → F2 截图** → 导出当场看到

## 问答预案

| 会问啥 | 回答 |
|---|---|
| OBJ 加载器处理了哪些格式？ | v/vt/vn + f 的三种格式（p, p/t, p/t/n, p//n），三角剖分 + 顶点去重 |
| NURBS 怎么实现无缝闭环？ | Periodic 模式：整数均匀节点向量 + 参数取模，curve C² 连续 |
| 阴影怎么做的？ | 两 pass：directional ortho depth map + spot perspective depth map，Blinn-Phong 里 PCF 3×3 采样 |
| 门检测怎么处理旋转？ | 射线变换到门局部空间做 AABB 检测，命中后弹簧阻尼动画驱动开关 |
| 后坐力怎么做的？ | 弹簧阻尼微分方程：F = -k·x - damping·v，逐帧欧拉积分 |
