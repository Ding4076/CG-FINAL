# AimTrainer 操作手册

> 三维瞄准训练游戏。快速上手 + 全部控制项 + 调试说明。

## 一、启动流程

1. 编译运行（见 README 或文末「构建运行」）。
2. 启动后先播一段**开场动画**（OBJ 序列：变形球，顶点逐帧变化），约 5 秒后自动进入游戏。
3. **跳过动画**：任意键（空格 / 回车 / Esc）或左键。
4. 进入游戏后默认 **FPS 模式**，鼠标自动锁定转视角，准星指向首个靶子。

## 二、核心玩法

| 模式 | 说明 |
|------|------|
| **Standard（标准）** | 单个静态气球靶，打中后立刻在视野内随机重生。限时计分。 |
| **Wild（狂野）** | 3 个球沿自研 NURBS 曲线运动，球-球互斥碰撞。更难，更乱。 |

模式切换：F1 面板 → Game → Standard / Wild（切换会重开本局）。

| 武器 | 说明 |
|------|------|
| **手枪 [1]** | 无限射速，小准星，无开镜。 |
| **狙击 [2]** | ~1 发/秒限速，**右键开镜**（FOV 缩小 + 圆形视野遮罩），无准星（靠镜瞄准）。 |

切武器：按 `1` / `2`，或 F1 面板选。

## 三、键鼠操作

| 按键 | 作用 | 备注 |
|------|------|------|
| **WASD** | 移动 | FPS 模式，仅水平面（不飞） |
| **鼠标移动** | 转视角 | FPS 模式；灵敏度 F1 可调 |
| **左键** | 射击 | FPS 模式；光标锁定时 |
| **右键** | 开镜 | 仅狙击，按住 |
| **1 / 2** | 手枪 / 狙击 | |
| **L** | 手电开关 | 跟随相机的聚光灯，照亮看的方向 |
| **Tab** | FPS ↔ Orbit 相机 | |
| **F** | 视角缩放到 Fit | 仅 Orbit |
| **F1 / Esc** | 暂停面板 | 冻结计时器；面板里 Resume / Exit |
| **F2** | 截图 | 随时，存 `media/screenshots/shot_XXX.png` |
| **空格 / 回车 / Esc / 左键** | 跳过开场动画 | 仅动画期间 |

## 四、暂停面板（F1 / Esc）

打开后**冻结计时器**，光标自由。逐项：

- **Resume Game (F1)** / **Exit Game**：继续 / 退出。
- **Controls**：鼠标灵敏度（0.01–0.5）。
- **Display**：Fullscreen 全屏切换。
- **Camera**：显示当前 FPS/Orbit；Orbit 下有「Zoom To Fit (F)」。
- **Directional Light**：开关 / 方向 / 颜色 / 强度 / 是否投影。
- **Point 0 / Point 1**：点光源开关 / 位置 / 颜色 / 强度。
- **Spot 0**：聚光灯（场景顶灯）开关 / 位置 / 方向 / 颜色 / 强度 / 锥角 / 是否投影。
- **Flashlight (camera) [L]**：手电开关 / 颜色 / 强度 / 锥角。
- **Hemisphere Ambient**：天空色 / 地面色（影响所有面的环境光，墙太暗就拉亮）。
- **Game**：模式、武器、限时、球大小、Wild 速度、Restart Round、当前命中/射击/命中率。
- **Capture / Export**：F2 截图提示；**Export scene to .obj** 导出当前场景。

> 调光即时生效。墙/物体太暗 → 拉高 Hemisphere Ambient 的 sky/ground 颜色，或开手电。

## 五、HUD（画面元素）

- **顶部居中倒计时**：0.01 秒精度。白→黄(<10s)→红(<3s)。限时设为 0 则不显示（无限制）。
- **准星**：手枪显示小十字；狙击未开镜无准星。
- **开镜遮罩**：狙击右键时，圆外区域变黑 + 十字线。
- **回合结束弹窗**：时间到自动弹出，显示命中 / 射击 / 命中率 / 排行榜（本局历史，最多 8 条）+ Play Again / Quit。

## 六、输出文件（运行时生成）

都写在 exe 同目录的 `media/` 下（即 `build/media/`）：

| 内容 | 路径 | 触发 |
|------|------|------|
| 截图 | `media/screenshots/shot_000.png` … | F2，自增编号 |
| 场景导出 | `media/exports/scene.obj` | F1 → Export scene to .obj |
| 开场动画帧 | `media/models/seq/frame_000.obj` …（30 帧） | 首次启动自动生成并读回 |

> 开场动画的 30 帧 OBJ 是程序首次运行时自生成的（变形球），之后从磁盘读回播放 —— 满足「读取连续 OBJ 文件多帧绘制」。

## 七、调试 / 演示技巧

- **不想被 60s 计时器打断**：F1 面板 → Game → time limit 拉到 **0**（无限制）；或随时 Esc 暂停。
- **看光照 / 阴影细节**：按 Tab 切 Orbit，右键拖转、滚轮缩、中键平移，F 自动框选场景。
- **看靶子轨迹（Wild）**：Orbit 模式俯视，能看见 3 条 NURBS 曲线运动 + 球互斥。
- **截图存证**：F2 随时存，演示/报告用。
- **导出当前场景**：F1 → Export，得到 .obj 给报告或外部工具看。

## 八、构建运行（MinGW，VSCode）

已配 `.vscode/`（本机用，不入库）：

- **一键**：`Ctrl+Shift+B`（任务 build and run，编译完自动开窗口）。
- **命令行**（终端）：
  ```bash
  cmake -B build -G "MinGW Makefiles" -S .
  cmake --build build -j 8
  cd build && ./FinalProject.exe
  ```
- **加新 .cpp 没生效**：Terminal → Run Task → `clean rebuild`。

> 工具链：g++ 15.2（MSYS2 MinGW64）、cmake 4.3、mingw32-make。
> 注意：PowerShell 直接跑 cmake 会因 Git 的 `sh.exe` 冲突失败，VSCode 任务已用干净 PATH 绕开。

## 九、常见问题

| 现象 | 原因 / 解决 |
|------|------|
| 启动黑屏 / 崩 | 多半 OpenGL 上下文；需支持 GL 3.3（Intel Iris Xe / NVIDIA / AMD 均可） |
| 没有 | 音效文件缺失 → 已内置 3 个合成 wav；若仍无声检查 `media/sounds/` 是否随 exe 拷贝 |
| 墙体太暗 | F1 → Hemisphere Ambient 拉亮 sky/ground，或开手电(L) |
| 截图没生成 | 必须从 `build/` 目录运行（exe 用相对路径读 `media/`） |
| 编译找不到新文件 | CMake GLOB 缓存，跑 `clean rebuild` 或重新 `cmake -B build` |
