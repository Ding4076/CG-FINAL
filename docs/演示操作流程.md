# SNAPFIRE 课堂演示操作流程

> 实机操作卡 · 约 2 分钟 · 照着走

## 准备工作

- [x] 已编译 `build\FinalProject.exe`
- [ ] 先打开游戏确认正常（避免演示时翻车）
- [ ] 把命令行 / 资源管理器保持在 build 目录，准备双击 exe

---

## 演示步骤

### 步骤 1：启动 + 开场动画（幻灯第 3 页）

- 双击 `build\FinalProject.exe`
- **等开场动画播完**——变形球，约 5 秒后自动进游戏
- **讲**："这是 OBJ 序列动画，30 帧网格逐帧变化，真几何动画"

### 步骤 2：手枪打靶（同一页）

- 进游戏后**直接用 LMB 打 2 个靶**
- **讲**："手枪无限连发，命中后粒子爆炸、右上角 +1"
- 注意：别打太快，让观众看清反馈

### 步骤 3：狙击开镜（同一页）

- **按 2** 切狙击
- **RMB 按住开镜** → LMB 打一枪
- **讲**："狙击约 1 发/秒，开镜 FOV 缩至 15°，镜外全黑"
- 松 RMB

### 步骤 4：后室彩蛋（同一页）⭐

- **WASD 走到后墙**（转身面向 -z 方向，看到木门）
- **LMB 射门** → 门自动弹开
- **走进去** → **LMB 打金色大靶**
- **讲**："功能交互真实性加分项——射击开门，弹簧阻尼动画，进后室 +3 分"
- WASD 走出门回到主战场

### 步骤 5：Wild 模式 + NURBS（幻灯第 7 页）

- **F1 / Esc 开面板**
- Game → 点 **Wild (NURBS)**
- **关面板**（再按 F1/Esc）
- **Tab 切 Orbit 相机**
- **鼠标右键拖转视角**，从上方俯视
- **讲**："三条 NURBS 曲线，球沿轨迹运动，无缝闭环不会跳"
- 让观众看几秒球的运动轨迹

### 步骤 6：阴影对比（幻灯第 8 页）

- **F1 开面板**
- 找到 **Directional Light**
- **取消勾选 Shadows (directional)** → 关闭面板（F1）→ 看地面影子消失
- **再开 F1，勾回 Shadows** → 关面板 → 对比
- **讲**："方向光 + 聚光灯双 Shadow Map，2048², PCF 3×3 软阴影，实时开关"
- Tab 切回 **FPS** 恢复第一人称

### 步骤 7：演示自由操作

- **F1 → Game → Standard** 切回标准模式（如果场景乱了）
- **开手电**：按 L，FPS 下往前走，看光照前方
- **F2 截图**：按一下（之后展示截图文件）

## 关键键位速查

| 键 | 功能 |
|------|------|
| LMB | 射击 |
| RMB | 狙击开镜 |
| 1 / 2 | 手枪 / 狙击 |
| Tab | FPS ↔ Orbit |
| L | 手电 |
| F1 / Esc | 暂停面板 |
| F2 | 截图 |

## 如果出问题

| 状况 | 怎么处理 |
|------|------|
| 开场动画一直不结束 | 按空格/左键 手动跳过 |
| 靶子打不中 | 瞄准球心，慢一点；或站近一点 |
| 切到 Wild 后球看不见 | Tab 切 Orbit + 右键转视角找一下 |
| 门打不开 | 走近一点，对着门扇中间打 |
| 进后室出不来了 | WASD 往坡道方向走（进门左前方有回程坡道） |
